# About

written by **Golang**

this tools depends on the **[gocui](https://github.com/jroimartin/gocui)**

and **[vim-keybind](https://gist.github.com/jroimartin/1ac98d3da7278fa18866c9cae0af6007)**

thanks!

# Usage

write `alias gv=<govim binarypath>` in `.bashrc` 

```sh
$gv <filepath>
```
